# blackBelt

Este projeto é uma loja virtual de celulares, focada na venda de produtos Apple. É uma aplicação web frontend estática, desenvolvida para apresentar produtos, informações sobre a loja e um carrossel de itens.

## Funcionalidades

-   **Página Inicial (Home)**: Boas-vindas e apresentação da proposta da loja.
-   **Sobre**: Informações detalhadas sobre a Black Belt.
-   **Galeria**: Exibição de modelos de celulares (lacrados e seminovos) com carrosséis interativos.
-   **Clientes**: Seção para depoimentos ou informações relacionadas a clientes.
-   **Contato**: Informações de contato.
-   **Navegação Responsiva**: Barra de navegação adaptável a diferentes tamanhos de tela.
-   **Efeitos Visuais**: Animações e transições para uma experiência de usuário dinâmica.

## Tecnologias Utilizadas

-   **HTML5**: Estrutura da página web.
-   **CSS3**: Estilização e design responsivo.
-   **JavaScript**: Interatividade, carrosséis de imagens e efeitos visuais.
-   **Font Awesome**: Ícones utilizados na interface.

## Pré-requisitos

Para visualizar e interagir com este projeto, você precisará apenas de um navegador web moderno (como Google Chrome, Mozilla Firefox, Microsoft Edge ou Safari).

## Instalação e Execução

Siga os passos abaixo para configurar e executar o projeto em seu ambiente local.

### 1. Clonar o Repositório

Abra seu terminal ou prompt de comando e execute o seguinte comando para clonar o repositório:

```bash
git clone https://github.com/RaphaLsantos/blackBelt.git
cd blackBelt
```

### 2. Abrir o Projeto no Navegador

Como este é um projeto frontend estático, você pode abri-lo diretamente no seu navegador web. Navegue até o diretório `user` e abra o arquivo `index.html`.

```bash
cd user
start index.html # No Windows
open index.html # No macOS
x-www-browser index.html # No Linux (pode variar dependendo da sua distribuição)
```

Alternativamente, você pode usar uma extensão de servidor local para VS Code (como `Live Server`) ou qualquer outro servidor HTTP estático para servir os arquivos. Isso é recomendado para garantir que todos os recursos (como imagens e scripts) sejam carregados corretamente, especialmente se você planeja desenvolver ou testar o projeto.

## Uso da Aplicação

Com o projeto aberto no navegador, você poderá navegar pelas diferentes seções da loja virtual, explorar os carrosséis de produtos e interagir com os elementos visuais.
